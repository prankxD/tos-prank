Peak Mic is a multi-purpose Discord bot designed to provide moderation, automoderation, music playback, logging, and server management tools. By inviting or using Peak Mic, you agree to these Terms of Service.

Bot Functionality

Peak Mic provides:
  • Moderation & Automoderation
  • Music Playback
  • Server Logging
  • Role Management
  • Channel Management
  • Anti-Spam & Anti-Raid Systems
  • Custom Server Configuration

Administrator Permission Requirement

Peak Mic may request Administrator Permission to ensure that moderation and automoderation systems function correctly. Administrator permission is required for:
  • Automoderation enforcement
  • Role assignment/removal
  • Timeout / mute system
  • Channel lockdown system
  • Anti-Raid protection
  • Message filtering & spam control
  • Logging moderation actions
Peak Mic will never perform destructive actions such as deleting channels, roles, or kicking members without an explicit moderation command from an authorized server administrator. Administrator permission is recommended but optional. Server owners may grant individual permissions manually if preferred.

Permissions Required

Peak Mic may require:
  • View Channels
  • Send Messages
  • Manage Messages
  • Manage Roles
  • Manage Channels
  • Connect & Speak (Music)
  • Read Message History
  • Use Voice Activity
Permissions are used strictly for: Moderation, logging, automoderation, and music functionality.

Privacy & Data Usage

Peak Mic may store:
  • Server ID

  • Moderation logs

Peak Mic does NOT store:
  • Email addresses
  • Passwords
  • IP addresses
  • Private user data
Stored data is used only for bot functionality and is never sold or shared.
