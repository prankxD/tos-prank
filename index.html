Bot Functionality

Peak Mic is a music bot designed to allow users to search for and play music within a Discord server environment. Proper functioning requires certain Discord permissions and intents:

  • Server Member Intent: Necessary to provide accurate functionality related to server members, such as managing member-specific commands and interactions.
  • Message Content Intent: Required to read messages for processing music commands and optionally delete commands after processing to maintain chat cleanliness.

Permissions Required

Peak Mic requires the following standard Discord permissions granted by a server administrator to operate effectively:

  • Read Messages / View Channels: To detect commands directed at the Bot.
  • Send Messages: To provide feedback, playback information, and respond to users.
  • Manage Messages: Optionally used to delete processed music commands for chat tidiness.
  • Connect: To join voice channels for music playback.
  • Speak: To stream audio (play music) in voice channels.
